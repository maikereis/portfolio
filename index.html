Motor de Alvos

Sistema de classificação para geração de escores de fraude em 28 regionais, otimizando a identificação de irregularidades em consumo de energia.

Detalhes do Projeto

Tecnologias: Docker, Python, Scikit-learn, Airflow, MLflow, OracelSQL, Optuna

Resultados:

  • Aumento em mais de 1000% na oferta de alvos.
  • Saindo de 1 para 28 regionais com modelos em produção.
  • Tempo entre para adaptação de modelos de dias para horas.

Tratamento de Denúncias com LLM

Sistema inovador utilizando LLMs como Agentes de IA para processamento e análise automatizada de denúncias de fraude.

Detalhes do Projeto

Tecnologias: Python, LangChain, OpenAI API, FastAPI

Resultados:

  • NER para extração de endereços
  • Engenharia de Prompt.
  • Economia de aproximadamente 78% no tempo de tratamento da denúncia.
